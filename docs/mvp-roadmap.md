# MVP Roadmap

## Phase 1

- Build the shared feature schema
- Implement web-page feature extraction
- Implement deterministic risk rules
- Return explainable warnings in Chrome Extension

## Phase 2

- Integrate production LLM provider
- Add prompt templates for phishing semantics
- Track confidence and attack-type classification

## Phase 3

- Add feedback capture
- Store audit trail and analyst labels
- Build evaluation datasets and dashboards

## Phase 4

- Add agent tools for external verification
- Add asynchronous deep-analysis workflow
- Add policy tuning and RL experimentation

## Engineering Priorities

- Keep the extension thin
- Keep all scoring explainable
- Separate deterministic and probabilistic logic
- Log enough evidence for future model improvement
